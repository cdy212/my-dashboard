3단계로 끝나는 간편한 사용법

1. 확장 프로그램 실행

Chrome 브라우저에서 Gofer 아이콘을 클릭하여 패널을 엽니다. 별도의 가입이나 복잡한 설정이 필요 없습니다.

2. 데이터 포인트 선택

수집하고 싶은 웹 요소를 마우스로 클릭하세요. Gofer가 자동으로 유사한 패턴의 데이터를 인식합니다.

3. 데이터 추출 및 활용

수집된 데이터를 실시간으로 확인하고, Excel(CSV) 또는 JSON 형식으로 즉시 다운로드하세요.

전문가를 위한 강력한 세부 기능

스마트 셀렉터 엔진

단순 클릭만으로 복잡한 CSS/XPath 셀렉터를 자동으로 최적화하여 생성합니다.

페이지네이션 자동 처리

'다음 페이지' 버튼을 인식하여 여러 페이지에 걸친 데이터를 한 번에 수집합니다.

다양한 내보내기 포맷

현업에서 가장 많이 쓰이는 CSV(엑셀 호환) 및 개발 친화적인 JSON 내보내기를 지원합니다.

동적 웹사이트 지원

React, Vue 등으로 만들어진 SPA(Single Page Application) 사이트의 데이터도 문제없이 수집합니다.
